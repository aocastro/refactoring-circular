## 2024-04-04 - Pagination ARIA Labels\n**Learning:** The PaginationControls component is highly reused across the application (in dashboard and admin sections). Adding ARIA labels for localized navigation strings ("Página anterior", "Ir para a página N", "Página seguinte") significantly improves the screen reader experience globally without requiring localized changes in each usage.\n**Action:** Always check globally reused components like pagination, navigation, or data tables for basic ARIA labels on icon-only buttons as they yield the highest a11y ROI.
